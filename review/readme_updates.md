# Proposed README Updates

This file accumulates proposed changes to README.md discovered during scientific code review.

Each review iteration may identify:
- Inaccuracies in documentation vs implementation
- Missing documentation for implemented features
- Scientific corrections needed
- Clarifications for edge cases

## Format

Each entry should include:
- **Module**: Which module the review was for
- **Date**: When the issue was found
- **Section**: Which README section needs updating
- **Current text**: What the README currently says (if applicable)
- **Proposed text**: What it should say
- **Rationale**: Why this change is needed
